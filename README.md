# AgoraAI

AgoraAI is a multi-agent AI discussion and debate platform built using FastAPI, WebSockets, and large language models. It enables multiple AI personas to participate in structured conversations, debates, and collaborative reasoning in real time.

The project is designed to explore agent orchestration, conversational memory, real-time communication, and persona-driven responses in a scalable backend architecture.

---

## Features

- Multi-agent AI system with configurable personas
- Real-time communication using WebSockets
- Structured debate and discussion flow
- Persistent logging of conversations and agents
- Modular backend architecture for easy extension
- Frontend interface for interacting with agents
- Secure environment variable management for API keys
